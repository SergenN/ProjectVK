1. Ga naar File --> package structure  (CTRL + SHIFT + ALT + S)
2. klik + -> Jar -> Empty
3. Hernoem de artifact VnK
4. Sleep ProjectVK van rechts naar links (hover op VnK folder om los te kunnen laten)
5. Klik op VnK.jar en klik dan op Create Manifest button onderaan en klik op OK
6. Selecteer de main class
7. Klik op + -> Extracted directory
8. Zoek naar de library files van xchar en OPEN DEZE selecteer dan de 'com' folder en klik op OK
9. doe hetzelfde voor de xchar-demo bestand
10. Verander de output directory naar bureaublad
11. Druk op apply en OK
12. Klik op Build in de navigatie balk (ALT+B)
13. Klik op Build artifacts --> VnK --> Build
--Open de jar door er op te klikken of:
Kopieer launch.bat naar het bureaublad en voer het uit.
(Hierbij krijg je dus ook het CMD console waar system.out messages en exceptions worden weergeven)
